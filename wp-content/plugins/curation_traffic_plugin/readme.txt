Easily add content curaiton to any WordPress site.

== Description ==

The Curation Traffic plugin allows you easily add curated content to any WordPress website or blog. With it the CurateThis content pulling bookmarklet it makes curating content push button simple.

== Installation ==

Install as usual and activate with the You Brand, Inc. Products Licensing Plugin. Please note Ioncube and PHP5.3 is required.
